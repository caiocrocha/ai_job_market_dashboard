# AI Job Market Dashboard

This project scrapes AI and data job postings from multiple sources, normalizes roles and skills, and visualizes market trends in an interactive Streamlit dashboard.

**Features**

* Multi-source job scraping
* Skill and role normalization
* Experience level aggregation
* Interactive charts (skills, roles, trends)

**Tech stack**
Python · Playwright · Hugging Face · Pandas · Plotly · Streamlit
